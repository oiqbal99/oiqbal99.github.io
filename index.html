SMS Butler


SMS-Butler is an android application which utilizes XML and java to provide a seamless user experience and provides automated services to reduce user interaction. SMS-Butler is designed to reply to incoming messages and calls with a user-generated message automated whenever the application is activated. The application allows users to store multiple responses allowing them to easily shuffle between automated responses.
